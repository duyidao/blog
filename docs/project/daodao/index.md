## 项目搭建

1. 创建项目
   ```
   npm init vite@latest
   ```
    
2. 安装依赖
   ```
   npm install
   ```
   依赖安装 `vue-router` 、`less` 、`less-loader` 等项目需要的依赖
   
3. 启动项目
   ```
   npm run dev
   ```
   
4. 打包项目
   ```
   npm run build
   ```
